# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project follows [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-20

### Added

- Added `Age` validation rule.
- Added `SNILS` validation rule with official checksum validation.
- Added `INN` validation rule for individuals (12 digits) and legal entities (10 digits).
- Added English and Russian translations.
- Added comprehensive PHPUnit test suite.
- Added PHPStan support.
- Added package documentation.
